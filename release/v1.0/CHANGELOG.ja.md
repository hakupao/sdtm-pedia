# 変更履歴

## v1.0 — 2026-04-27

初回公開リリース。

### 追加

- SDTM Pedia v1.0 ナレッジベース文書を公開。
- Claude Projects、ChatGPT GPTs、Gemini Gems、NotebookLM の利用入口を追加。
- 中国語、英語、日本語のユーザー文書を追加。
- プラットフォーム選択、例示質問、用語集、既知の制約、方法説明を追加。
- 自分でインスタンスを設定する管理者向けガイドを追加。

### 内容範囲

- よくある SDTM ドメイン、変数、統制用語、ドメイン境界、クロスドメイン関係の確認に対応。
- 存在しない変数、廃止済み概念、不適用な経路など、よくある誤前提に慎重に対応。
- 大規模な用語セット、リアルタイム外部情報、組織固有ルールについて明確な境界を設定。

### 利用上の注意

- 本ツールは検索と初期確認を支援するものであり、CDISC 公式刊行物を置き換えません。
- 正式提出、医学コーディング、プロジェクトレベルのマッピング、品質管理は内部手順に従ってください。
- 不足や不整合を見つけた場合は、質問、プラットフォーム、回答、期待される根拠を記録し、メンテナーに共有してください。
