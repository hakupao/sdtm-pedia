# 更新日志

## v1.0 — 2026-04-27

首个公开发布版本。

### 新增

- 发布 SDTM Pedia v1.0 知识库说明与使用文档。
- 支持 Claude Projects、ChatGPT GPTs、Gemini Gems 和 NotebookLM 四类使用入口。
- 提供中文、英文、日文三语用户文档。
- 提供平台选择、示例问题、术语表、已知限制和方法说明。
- 为需要自行配置实例的用户提供管理员部署指南。

### 内容范围

- 覆盖常见 SDTM 域、变量、受控术语、域边界和跨域关系查询。
- 支持对常见错误前提进行谨慎处理，例如不存在的变量、已弃用概念或不适用路径。
- 对大型受控术语、实时外部资料和组织内部规则保留明确边界。

### 使用提醒

- 本工具用于辅助查询和初步复核，不替代 CDISC 官方出版物。
- 正式提交、医学编码、项目级映射和质量控制仍应遵循组织内部流程。
- 如发现内容缺口或不一致，请记录提问、平台、回答和期望依据，反馈给维护者。
