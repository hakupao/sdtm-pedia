# Example Questions

This neutral page is retained for compatibility. For the website, use the language-specific versions named `DEMO_QUESTIONS.zh.md`, `DEMO_QUESTIONS.en.md`, and `DEMO_QUESTIONS.ja.md`.

The questions are intended to help users learn how to ask SDTM Pedia for useful, reviewable answers. They are not a formal validation plan.
