# Disposition Codelists

> Codelists for disposition and protocol deviation domains.

## Category of Disposition Event (C74558)

Extensible: No

| Code | CDISC Submission Value | CDISC Synonym(s) | CDISC Definition |
|------|----------------------|-------------------|------------------|
| C74590 | DISPOSITION EVENT | Disposition Event | The group of incidents that occur during a clinical trial and describe whether a subject completed the study epoch or the reason this event did not occur. The subject's disposition is often described for each phase of the study. |
| C150824 | OTHER EVENT | Other Event | Other important events that occur during a trial but are not driven by protocol requirements. |
| C74588 | PROTOCOL MILESTONE | Protocol Milestone | Occurrences within some epochs that are initiated by a protocol-specified event. The most common protocol milestone is "Informed Consent Obtained" which usually documents the start of the study. |
