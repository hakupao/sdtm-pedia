### 6.2 预防规则 (7 条)

来源: `meta/retrospective.md` (规则 1-4) + Issue 3 分析 (规则 5-7)

| # | 规则 | 类型 |
|---|------|------|
| 1 | 定量 PASS 标准 (覆盖率 ≥ 95%, 零占位标记, 行数/页数比合理) | 必须 |
| 2 | 写/审必须分离 (补写 agent ≠ 复核 agent, 不共享上下文) | 必须 |
| 3 | AI 估算值必须标记 (如 PDF 页码标注 estimated/±N) | 必须 |
| 4 | 人工抽样检查点 (每阶段用户随机抽查 3-5 样本) | 强烈建议 |
| 5 | Agent 拆分必须按章节边界，不按页码均分 | 必须 |
| 6 | Prompt 中必须列出完整子节清单 | 必须 |
| 7 | 修复任务目标是"确保完整覆盖"，不是"修复 N 个已知问题" | 必须 |
