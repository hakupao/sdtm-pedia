## 题文 (SMOKE_V4 §2 Q1)

某肿瘤试验对受试者外周血样进行 EGFR 基因测序, 在 Exon 19 位置发现一个已知的激活突变 (dbSNP rs121913444, 导致 L858R 氨基酸替代). 这条结果应该记录在 SDTMIG v3.4 的哪个域? 请列出该记录至少 **5 个 Core=Req 变量** 和 **3 个 Core=Exp 变量** (按 Topic 变量优先), 并说明: (a) 如何记录"Exon 19"位置信息; (b) 如何引用 dbSNP ID; (c) 基因组参考版本 (比如 GRCh38.p13) 存哪; (d) 如果该变异可遗传给下一代, 走哪个变量.

## 原始 AI 回答 (整理)

应记录在 GF (Genomics Findings, 基因组学发现) 域. 该域用于核酸层面的遗传/基因组结果, 也可包含由核酸结果推断的蛋白/氨基酸变化; 结构是 one record per finding per observation per biospecimen per subject.

GF 示例中同类 SNV 结果就是用 GFTESTCD=SNV, GFTEST=Single Nucleotide Variation, 并用 GFTSTDTL 区分 "PREDICTED AMINO ACID CHANGE / VARIANT CLASSIFICATION" 等结果细节.

**至少 5 个 Core=Req 变量**:
| 变量 | 说明 |
|---|---|
| STUDYID | Study Identifier |
| DOMAIN | GF |
| USUBJID | 受试者唯一 ID |
| GFSEQ | GF 域内序号 |
| GFTESTCD | SNV, 若按单核苷酸变异记录; Core=Req, Role=Topic |
| GFTEST | Single Nucleotide Variation; Core=Req |

**至少 3 个 Core=Exp 变量**:
| 变量 | 建议值/说明 |
|---|---|
| GFORRES | 原始结果, 例如 p.Leu858Arg 或报告原文 L858R |
| GFSTRESC | 标准化字符结果, 例如 p.Leu858Arg |
| GFMETHOD | 测序/检测方法, 例如 NEXT GENERATION SEQUENCING |
| GFREFID | 样本/检测参考 ID |

**4 个关键问题**:
| 问题 | 变量 | 记录方式 |
|---|---|---|
| (a) Exon 19 位置信息 | **GFGENSR** | 存 "Exon 19". Genetic Sub-Region, 用于记录变异所在的基因亚区域 |
| 更精确的坐标位置 | GFGENLOC | 若有基因组坐标 |
| (b) dbSNP ID | **GFPVRID** | 存 "rs121913444". Published Variant Identifier |
| (c) 基因组参考版本 | **GFGENREF** | 存 "GRCh38.p13". Genome Reference |
| (d) 可遗传给下一代 | **GFINHERT** | 存受控术语值如 "GERMLINE VARIATION". **注意变量名是 GFINHERT, 不是 GFINHERTG**. 该变量对应 **CT C181177**; codelist 包含 GERMLINE VARIATION / MITOCHONDRIAL VARIATION / SOMATIC VARIATION |

示例骨架 (extra): DOMAIN=GF, GFTESTCD=SNV, GFTEST=Single Nucleotide Variation, GFTSTDTL=PREDICTED AMINO ACID CHANGE, GFORRES=p.Leu858Arg, GFSYM=EGFR, GFSYMTYP=GENE WITH PROTEIN PRODUCT, GFGENSR=Exon 19, GFPVRID=rs121913444, GFGENREF=GRCh38.p13, GFINHERT=GERMLINE VARIATION (若可遗传) 或 SOMATIC VARIATION (肿瘤来源).
