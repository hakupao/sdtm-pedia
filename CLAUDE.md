## Session Startup

Every new session, **before doing any work**, read these files in order:

1. `.work/MANIFEST.md` вҖ” file layout, change chains, quick reference table
2. `.work/meta/worklog.md` вҖ” work log with recovery guide
3. `.work/03_verification/issues_found.md` вҖ” open issues
4. `.work/meta/retrospective.md` В§ 4 вҖ” four prevention rules (must follow when doing any AI-assisted content work)

Then summarize to the user: current status, open issues, and suggested next step.
