# old_knowledge_base_v0/ — 早期中文目录版 SDTM 知识库 (已归档)

> 归档日期: 2026-05-06
> 原位置: `project_knowledge_base/` (repo 顶层)
> 取代者: `knowledge_base/` (英文版, 当前唯一活跃 KB)

## 是什么

这是项目早期的一版中文目录结构 SDTM 知识库, 按"核心 Domain 变量 / 完整规范文本 / Controlled Terminology / 其余 Domain 变量"4 大类组织。

## 为什么归档

- 当前活跃的 `knowledge_base/` 改用英文目录 + Domain 维度组织 (chapters/ + domains/ + model/ + terminology/), 与 CDISC 官方源结构一致, 便于双向 traceability
- 早期版本作为方法论演进的参考保留, 不再投入维护
- 内容上与 `knowledge_base/` 大量重叠, 主项目流程不再引用此处

## 何时回查

- 想看"早期是怎么组织的", 复盘方法论演进
- 某个细节在当前 KB 不确定时, 交叉对比早期版本写法

## 何时不应回查

- 任何对外交付 / API / 索引 / agent 检索 — 一律走 `knowledge_base/`
- 任何内容修订 — 改 `knowledge_base/`, 不改这里
